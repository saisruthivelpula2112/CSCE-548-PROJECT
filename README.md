# CSCE 548 – Project 1

## Project Overview
This project demonstrates database design and implementation using a relational database.
It includes:
- A normalized database schema with at least 5 tables
- Primary keys and foreign key relationships
- Seed data with more than 50 rows
- A Python data access layer
- A console-based application to interact with the database

## Technologies Used
- PostgreSQL
- Python
- SQLAlchemy
- GitHub
